### Commands ###

 - Interface Settings

```bash
vi /etc/network/interfaces
```

 - Manually Disable / Enable Inferace

```bash
ifdown eth0
ifup eth0
```

 - DNS Client Settings

```bash
vi /etc/network/interfaces
```

 - Restart Networking

```bash
service network-manager restart
```
